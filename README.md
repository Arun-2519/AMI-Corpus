# AMI Meeting Summarization using AI

##  Project Overview
This project builds an AI-based meeting summarization system using the AMI Meeting Corpus.
It generates both extractive and abstractive summaries from multi-speaker meeting transcripts.

##  Models Used
- TF-IDF (Extractive Summarization)
- FLAN-T5 / BART (Abstractive Summarization)
